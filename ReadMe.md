### **Installation**
Required Package:
Flask: 2.2,
MatplotLib: 3.7.1,
Requests: 2.28.2
```
pip install flask==2.2
pip install matplotlib==3.7.1
pip install requests==2.28.2
```
------first install this package:
```
pip install -e .
```
-----Then to Run Application run this:
```
flask --app reserver run
```
